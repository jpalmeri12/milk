Hello!
This is MILK: My Interface for Learning Kanji. I made it because I couldn't find a kanji drawing app that has a spaced repetition system I liked. Also I really like milk which is where the name came from.

This web app is intended for use with the Kodansha Kanji Learner's Course. If you don't have it, I recommend getting the book. For each kanji you learn, read the entry in the book, and then write the kanji (preferably using a stylus on a tablet) until you've memorized it.

This app uses a spaced repetition system (SRS) because it seems to work for me, so you get to use it too. Let's get started!
